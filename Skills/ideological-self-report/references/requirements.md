# 思想汇报硬性要求

## 格式要求

### 页面设置
- 标题：居中写"思想汇报"
- 称谓：顶格写"敬爱的党组织："（在标题下一行书写，后加冒号）
- 正文：每段首行空两格
- 结尾："恳请党组织给予批评、帮助"
- 此致敬礼："此致"空两格，"敬礼"顶格
- 落款：居右写"汇报人：×××"，下一行写"×年×月×日"

### 字数要求
| 章节 | 字数限制 |
|------|----------|
| 总体报告 | 1700-1900字 |
| 思想政治方面 | 600-700字 |
| 学习方面 | 250-350字 |
| 工作方面 | 250-350字 |
| 生活/纪律方面 | 100-150字 |
| 缺点与不足 | 150-200字 |

## 内容要求

### 思想政治方面
- 结合最新时政热点
- 体现政治理论学习成果
- 联系自身实际谈认识
- 党章引用规范：涉及党章的内容必须跟党章对应内容严格一致，要抄全，抄到句号才能结束。（参考：[中国共产党章程](cpc-constitution.md)）
- 政治理论表述完整：提到马克思列宁主义、毛泽东思想、邓小平理论、"三个代表"重要思想、科学发展观时，必须同时提及习近平新时代中国特色社会主义思想，且顺序不能错

### 学习方面
- 体现专业知识学习
- 可包含考试、竞赛、论文等内容
- 突出学以致用

### 工作方面
- 具体工作职责和成果
- 服务意识和奉献精神
- 模范带头作用
- 在工作中应体现个人特色，结合本季度经历的事情

### 缺点与不足
- 真实具体，不空泛
- 每季度不重复，不能与之前写过的有超过一行一模一样的话
- 体现改进决心
- 禁止写"做事不够大胆"等模糊表述
- 每个季度写两三行左右即可
- 可写：理论联系实际、体能训练等方面

## 禁止事项

1. 禁止编造虚假信息
2. 禁止泄露国家秘密或敏感信息
3. 禁止抄袭他人内容
4. 禁止使用未经确认的政治表述
5. 禁止时空穿越：不能写当前时间点之后发生的事情
6. 禁止使用"自从被评为…"：应改为"自从被确定为…"
7. 禁止写"以党员的标准要求自己"、"党性修养"
