# 中文 AI 腔词表

处理中文文本时，优先删除或改写这些高频套话、翻译腔、黑话和空泛修饰。不要机械替换，先判断上下文，再改成具体、自然的中文。

## 过渡垫话

这些词通常只是在拖延进入主题。能删就删。

- 在当今瞬息万变的时代
- 在这个信息爆炸的时代
- 随着社会的不断发展
- 随着科技的不断进步
- 值得注意的是
- 不难看出
- 显而易见
- 事实上
- 总而言之
- 综上所述
- 总的来说
- 从某种意义上说
- 某种程度上
- 可以说
- 毫无疑问
- 不可否认的是
- 需要指出的是
- 我们不妨先来看看
- 接下来让我们一起

## 万能开头

这些开头会让文章像模板。直接写核心信息。

- 关于这个问题，我们需要从多个维度展开分析
- 要理解这个问题，首先要回到它的底层逻辑
- 这不是一个简单的问题
- 这个问题的答案并不唯一
- 在深入探讨之前，我们先明确一个概念
- 对于很多人来说，X 已经成为不可或缺的一部分
- X 正在深刻改变我们的生活和工作方式
- X 已经成为时代发展的重要趋势

## 翻译腔

中文里不需要保留英文句式的壳。

| 避免 | 改法 |
| --- | --- |
| 对于……而言 | 对……来说；或直接删掉 |
| 关于……的问题 | ……这件事；这个问题 |
| 通过……来实现 | 用……；靠……；直接写动作 |
| 作为一个……，他…… | 他是……，所以……；或直接写事实 |
| 这使得……成为可能 | 这样就能…… |
| 这意味着…… | 也就是说……；或直接写结果 |
| 其核心在于 | 关键是 |
| 在……的过程中 | ……时 |
| 基于……的考虑 | 考虑到…… |
| 从……角度来看 | 从……看；或删掉 |

## 大厂黑话

除非用户明确要求商业汇报、公关稿或行业黑话风格，否则换成普通话。

| 避免 | 可改成 |
| --- | --- |
| 赋能 | 帮助；让……能…… |
| 闭环 | 做完；形成完整流程 |
| 抓手 | 方法；入口；动作 |
| 颗粒度 | 细到什么程度 |
| 底层逻辑 | 原因；关键机制 |
| 深度链接 | 建立关系；打通 |
| 引爆 | 带火；拉动 |
| 点燃 | 激发；带动 |
| 矩阵 | 一组；一套 |
| 生态 | 一整套服务；相关参与方 |
| 沉淀 | 留下；积累 |
| 对齐 | 说清楚；达成一致 |
| 拉通 | 打通；一起处理 |
| 复用 | 重用；继续用 |
| 协同 | 一起做；配合 |
| 提效 | 提高效率；更快 |
| 降本增效 | 省钱、提速 |
| 方法论 | 方法；经验 |
| 心智 | 认知；印象 |
| 场景化 | 放到具体场景里 |

## 空泛修饰

这些词多数不能增加信息。删除后句子更干净。

- 非常
- 极其
- 相当
- 十分
- 显著
- 充分
- 有效
- 持续
- 全面
- 深入
- 重要
- 核心
- 关键
- 高质量
- 多维度
- 系统性
- 精准
- 充分发挥

保留规则：如果删掉后信息不变，就删。如果确实要表达程度，用具体数字、范围、对比或事实。

## 客服式结尾

这些结尾会削弱文本的真实感。

- 希望这篇文章能帮到你
- 希望以上内容对你有所启发
- 如果你也有类似困惑，不妨试试看
- 让我们一起期待
- 未来可期
- 相信在不久的将来
- 只有这样，我们才能……
- 这不仅是……，更是……

## 情绪化升华

不要用空泛情绪替代具体判断。

- 令人振奋的是
- 充满无限可能
- 开启新的篇章
- 迈向更加美好的未来
- 注入强劲动力
- 释放巨大潜能
- 展现出蓬勃生机
- 具有里程碑意义

## 判断方法

看到可疑短语时，问三个问题：

1. 删掉后信息是否减少？
2. 能不能换成具体动作、对象、数字或结果？
3. 真实的人聊天、写邮件、做说明时会不会这样说？

如果三个答案分别是“不减少”“能”“不会”，就删掉或重写。
