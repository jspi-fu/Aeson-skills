# 中文结构问题

中文 AI 腔不只体现在词语，也体现在段落和句式。处理中文文本时，优先识别下面这些结构，再重写成自然、具体的表达。

## 机械总分总

常见模式：

- 开头先讲背景和意义
- 中间用“首先、其次、最后”列三点
- 结尾再“综上所述”升华

问题：读者看到结构就能猜到后文，信息密度低。

改法：直接从最重要的信息写起。需要列点时，只列真正并列的事项，不为了凑结构补第三点。

## 强行三段式

常见模式：

- 不仅……还……更……
- 既能……也能……还能……
- 从 A、B、C 三个维度来看
- 第一是认知，第二是方法，第三是行动

问题：为了节奏堆内容，常把弱信息也塞进去。

改法：保留最有用的一到两点。能用一句话说清，就不要拆成三层。

## 正反对比套话

常见模式：

- 这是机遇，也是挑战
- 凡事都有两面性
- 这是一把双刃剑
- 一方面……另一方面……
- 它不是……而是……

问题：看似客观，实际没有说清楚具体利弊。

改法：直接写具体变化。比如不要写“AI 是机遇也是挑战”，改成“AI 能省掉重复劳动，也会让普通模板文案更快贬值”。

## 宏大背景开场

常见模式：

- 在当今时代……
- 随着技术的发展……
- 在数字化浪潮席卷全球的背景下……
- 近年来，X 越来越受到关注……

问题：开场太远，读者要等很久才知道你想说什么。

改法：第一句写结论、发现、问题或动作。背景只在必要时补一句。

## 空泛论证链

常见模式：

- X 很重要，因为它能提升效率、优化体验、促进发展
- 这背后反映了深层次的问题
- 它的核心价值在于构建长期能力

问题：每个词都像有意义，但没有具体内容。

改法：写清谁做了什么、改变了什么、结果是什么。不要用抽象名词互相解释。

## 虚假代理

常见模式：

- 这份报告揭示了……
- 数据告诉我们……
- 市场给出了答案
- 文化正在发生变化
- 这个产品重新定义了……
- 这篇文章将带你……

问题：抽象物被写成人，实际行动者被藏起来。

改法：能写人就写人。不能写具体人，就写“从数据可以看出”“用户开始……”“团队发现……”。

## 翻译式长句

常见模式：

- 句子里嵌套多层“的”
- 大量使用“对于、关于、通过、基于、围绕”
- 主语很长，谓语很晚才出现

问题：中文读起来绕，像英文句子直译。

改法：拆句。把动词提前。一个句子只承载一个主要意思。

## 客服式收尾

常见模式：

- 希望以上内容能给你带来帮助
- 如果你也有类似问题，不妨从今天开始行动
- 未来，让我们共同期待

问题：像客服话术，不像真实作者。

改法：在最后一个具体信息处停下。必要时给明确下一步，不要泛泛祝愿。

## 标题党式排比

常见模式：

- 从 0 到 1，打通增长闭环
- 看懂 X，抓住 Y，赢在 Z
- 不止 A，更是 B
- 一文讲透

问题：标题像卖课或营销号，降低可信度。

改法：标题直接说明内容、对象和结论。少用夸张动词。

## 改写顺序

1. 先删掉背景铺垫和总结升华。
2. 找出真正要说的结论、事实或动作。
3. 把抽象名词改成具体人、事、动作和结果。
4. 拆掉长句，减少三段式和排比。
5. 最后检查词语，把黑话和空泛修饰换掉。
